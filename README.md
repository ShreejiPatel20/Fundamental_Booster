# Fundamental Booster Submission

This repository contains the submission file for the Fundamental Booster assignment.

## Files

- `venv/Submission_01.py` - the assignment submission script.

## Notes

- `venv/` is a Python virtual environment directory.
- The real submission file is `venv/Submission_01.py`.

## How to use

1. Activate the virtual environment if needed.
2. Run the script with:
   ```bash
   python venv/Submission_01.py
   ```
